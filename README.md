# Kiria Arts Gallery 🎨

A beautiful, fully-responsive e-commerce platform showcasing traditional Indian art, Tanjore paintings, resin art, and handcrafted treasures.

![Kiria Arts](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## 🌟 Features

- **Art Gallery** - Browse curated collection of original Tanjore paintings and resin art
- **E-commerce Cart** - Full shopping cart functionality with quantity management
- **Art Classes** - Workshops for Tanjore Painting, Resin Art, and Mysore Painting
- **WhatsApp Integration** - Direct inquiry and checkout via WhatsApp
- **Responsive Design** - Beautiful UI with smooth animations
- **Image Zoom** - Click any artwork to view in full resolution
- **Search & Filter** - Find artworks by category, medium, and price
- **Customer Reviews** - Featuring real Google reviews with carousel
- **Contact Form** - Integrated Google Maps location

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (Framer Motion successor)
- **Icons**: Lucide React
- **Deployment**: Vercel

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/kiria-art-gallery.git
   cd kiria-art-gallery
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## 📦 Build for Production

```bash
npm run build
npm run preview
```
